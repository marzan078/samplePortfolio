AR Human Anatomy Learning App

A mobile Augmented Reality (AR) application designed to enhance anatomy learning through interactive 3D models. Built using Unity and Vuforia, the app includes system-based visualizations and a quiz feature to reinforce learning for students.

Main Menu Interface

Displays the enhanced anatomy AR app's main screen with quick access to core features like Tutorial, Laboratory, and Settings.

Immersive 3D Laboratory

A detailed virtual lab environment showcasing 3D models and workspace in AR.

Interactive Anatomy NPC

An engaging virtual character in the 3D lab that shares random, informative facts about human anatomy.

3D Body System Overview Panel

A user interface panel displayed in front of the 3D model providing a concise summary of the selected body system.

AR Camera View

A real-time augmented reality camera display showcasing the interactive 3D anatomical model.

Interactive 3D Model Exploration

User taps on a specific anatomical part, triggering highlight, descriptive panel display, and synchronized AI voice narration.

Clickable Parts Dropdown Panel

A user-friendly dropdown menu listing all interactive parts of the 3D model for easy navigation and selection.

Tutorial Interface

A simple navigation panel with three buttons guiding users through Laboratory, AR, and Quiz tutorials for seamless onboarding.

Step-by-Step Tutorial

An interactive tutorial UI allowing users to navigate instructions via swipe gestures, carousel thumbnails, or side arrow buttons.

Developer Profile

Featuring a team member’s profile, highlighting individual expertise and contributions to the collaborative 3D anatomy project.

Quiz Interface

An interactive quiz module designed to test users’ knowledge of human anatomy with engaging questions and immediate feedback.

QR Code Download Panel

A convenient pop-up displaying a QR code for users to easily download the 3D anatomy application or related resources.
